Predicting Product Weights at Self-Checkout Using Machine Learning

Self-checkout machines often fail when the actual weight of produce doesn’t match the system’s expected value. This leads to frustration and errors during checkout.I built a supervised machine learning model to predict the weight of products more accurately, reducing false alerts and improving user experience.

Status:Done Type: ML Year: 2024
Read Paper

At a glance

Tools

Python · Pandas · NumPy · Scikit-learn

Timeframe

2–3 weeks

Role

Solo project

Approach

  • Used linear regression as the core model.
  • Features included product surface area (measured by scanner) and apple category (e.g., Gala, Honeycrisp, Red Delicious).
  • Applied one-hot encoding for categorical features.
  • Implemented with Python, NumPy, Pandas, and Scikit-learn.
  • Evaluated using MSE, RMSE, and MAE metrics.

Outputs

  • Achieved MAE and RMSE ≈ 6.5 grams → predictions close to actual weights.
  • High R² score showed strong fit of the regression model.
  • Demonstrated feasibility of integrating ML into everyday retail technology.

Future Improvements

  • Use larger datasets for better accuracy.
  • Extend beyond apples to multiple produce types.
  • Consider hardware capabilities to measure the surface area and weight.
