# 📂 Personal Site (`gkm2004.github.io/personal-site`)

> "This is just a little bit of me on the internet."

Welcome to the repository for my personal site. It serves as a digital porch, a public notebook, and a grounding space for my thoughts, projects, and collaborations. 

## 🪵 Philosophy & Aesthetic
Instead of a glossy, over-engineered tech template, this space is designed with an **indie-web** and **junk journal** mindset. It’s meant to feel tactile, evolving, and human. 

Right now, it is intentionally simple. I am currently focusing my energy on shipping larger technical projects, so this site is a quiet work-in-progress—built to grow naturally over time rather than all at once.

## 🛠️ Built With
To keep it fast, lightweight, and permanent, this site uses no heavy frameworks or bloated dependencies:
* **HTML5** for structure
* **CSS3** for layout and custom styling
* **GitHub Pages** for hosting

## 📝 Sections Inside
* **Home / About:** A brief introduction to how I see the world through data, systems, and real-world contexts.
* **Projects / Collaborate:** Spaces highlighting shared learning and early-stage collaboration.
* **Binder & Journal:** A place to think in public, log notes, and keep an evolving record of reflections.

## 📌 Future Additions (When time allows)
When I'm not deep in the weeds of larger builds, I plan to inject more interactive and fun elements here:
- [ ] More custom, tactile UI elements (stickers, textures, hand-drawn accents).
- [ ] RSS feed integration for the Binder/Journal.
- [ ] A dynamic archive for small web experiments.
