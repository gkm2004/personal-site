prism — deployment risk intelligence
★ Grand Prize Winner Python Multi-agent DevOps

PRism

Agentic pre-deployment risk gate that evaluates pull requests before release and assigns a Deployment Confidence Score from 0–100. Four specialist agents run in parallel — Diff Analyst, History Agent, Coverage Agent, Timing Agent — and a Verdict Agent aggregates weighted signals. Built a GitHub Actions workflow integration and VS Code extension. Won the Grand Prize at Microsoft AI Dev Days Hackathon 2026.

csiro image2biomass — vegetation estimation
GeoAI Computer Vision Satellite Imagery

CSIRO Image2Biomass

ConvNeXt-based pipeline for estimating vegetation biomass from high-resolution satellite imagery. Framed as a robust regression task with fold-based validation, custom loss design for high-variance environmental data, and targeted augmentation informed by biome-level error analysis.

Write-up Kaggle
the lake project — sentinel-2 remote sensing
Remote Sensing GIS Python

The Lake Project

End-to-end Python remote sensing pipeline for temporal surface mapping of Sambhar Salt Lake using Sentinel-2 multi-spectral imagery. Handles cloud masking, spectral index calculation (NDVI, NDWI), and surface change analysis over time.
